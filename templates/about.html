About AgroDrone Navigator Project


Drone information can substantially complement the precision agriculture. To analyze this information recognition system required images, coordinate collection system and system displaying received data.

We built the recognition system with using machine learning algorithms. We extract information about coordinates from photo metadata. Received data (point with coordinates and what there are: norm, weeds, non-emergence) we export to a file that can be loaded to any agronavigator.
